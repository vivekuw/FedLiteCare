# Hospital A Uploads

Place local training or prediction CSV files in this folder.

The default training file expected by the config is `hospital_a_diabetes.csv`.

Use `hospital_a_diabetes.csv` for the split-dataset workflow.

Use `hospital_a_prediction_sample.csv` to try CSV-based prediction from the GUI.

Extra demo files:
- `hospital_a_followup_screening.csv`
- `hospital_a_weekly_screening.csv`
