# Test Input Samples

Sample CSV inputs for quick prediction checks live here.

Available files:
- `diabetes_prediction_sample.csv`
- `diabetes_test_low_risk_batch.csv`
- `diabetes_test_high_risk_batch.csv`
- `diabetes_test_mixed_cases.csv`
- `diabetes_test_borderline_cases.csv`
- `diabetes_test_followup_cases.csv`

These files use the same 8 diabetes screening fields expected by the hospital app.
You can upload any of them into `Hospital_A/uploads`, `Hospital_B/uploads`, or `Hospital_C/uploads`
and then use the `Patient Screening` tab to score the selected CSV.
